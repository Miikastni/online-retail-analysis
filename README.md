#  Анализ продаж интернет-магазина (Online Retail)

##  Описание проекта

В рамках проекта проведён анализ данных интернет-магазина с целью выявления ключевых факторов, влияющих на выручку, а также изучения поведения клиентов и сезонности продаж.

##  Цели проекта

* Рассчитать основные бизнес-метрики (выручка, количество заказов, средний чек)
* Определить наиболее прибыльные товары и клиентов
* Проанализировать динамику продаж во времени
* Выявить ключевые драйверы роста выручки

##  Предобработка данных

В ходе анализа данные были очищены:

* Удалены записи с отрицательным и нулевым количеством товаров (`Quantity <= 0`)
* Удалены записи с нулевой или отрицательной ценой (`UnitPrice <= 0`)
* Исключены некорректные транзакции (возвраты и ошибки)

##  Основные метрики

* **Выручка** = Quantity × UnitPrice
* **Количество заказов** = количество уникальных InvoiceNo
* **Средний чек (AOV)** = Выручка / Количество заказов

##  Основные результаты

### Сезонность продаж

* Пик продаж наблюдается в период **сентябрь–ноябрь**
* Снижение активности — в **феврале и апреле**

### Драйверы выручки

* В период сентябрь–ноябрь рост выручки обусловлен увеличением количества заказов
* В декабре наблюдается снижение количества заказов, однако рост среднего чека компенсирует это и поддерживает выручку

### Поведение клиентов

* Небольшая доля клиентов генерирует значительную часть выручки (эффект Парето)

##  Рекомендации

* Усилить маркетинговую активность в пиковые месяцы (сентябрь–ноябрь)
* Использовать стратегии увеличения среднего чека (бандлы, акции) в декабре
* Проанализировать причины снижения продаж в феврале и апреле
* Развивать работу с постоянными клиентами (retention)
