Анализ продаж интернет-магазина (Online Retail)

Задача:
Понять, за счёт чего растёт выручка и как ведут себя клиенты.

Что сделал:

1. Рассчитал ключевые метрики:
- выручка
- количество заказов
- средний чек

2. Проанализировал динамику продаж:
- по месяцам
- выявил сезонность

3. Исследовал поведение клиентов:
- распределение выручки по клиентам
- выявил эффект концентрации (Pareto)

Ключевые выводы:

- Выручка растёт в основном за счёт увеличения числа заказов, а не среднего чека
- В декабре поведение меняется: заказов меньше, но средний чек выше
- Небольшая доля клиентов приносит основную часть выручки
- Есть выраженная сезонность (пик осенью, спад в начале года)

Что это значит для бизнеса:

- Рост зависит от привлечения новых заказов → важно масштабировать трафик
- В декабре эффективнее работать через повышение среднего чека (upsell, bundles)
- Есть потенциал в работе с лояльными клиентами (retention)

Инструменты:
SQL (PostgreSQL), DBeaver
